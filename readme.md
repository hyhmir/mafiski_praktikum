# Mafiski praktikum

Moja porocila za mafiski praktikum. Naredi z njimi kar zelis. Imej lep dan.

## Uporaba
Odpri svoj najljubši terminal in vanj vpiši naslednje

```bash
git clone https://github.com/hyhmir/mafiski_praktikum.git

cd mafiski_praktikum

# optional: python –m venv myenv

pip install -r requirements.txt

cd rust

maturin develop --release
```

in nato po mili volji poganjaj katerokoli kodo :)
